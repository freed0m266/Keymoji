# 27 — Real app icon (před App Store releasem)

**Status:** Todo

**Priorita:** Pre-App-Store · **Úsilí:** S (designerská práce externí) · **Dopad:** High pro App Store

## Souhrn

V1.0 má placeholder ikonu (z Template scaffoldu nebo trivial generation). Před App Store submission potřeba reálnou.

## Scope (až přijde čas)

- Designerský brief: „klávesa K" nebo abstraktní keyboard motif, recognizable na home screen.
- Export: 1024×1024 master + všechny iOS icon sizes (Asset Catalog AppIcon.appiconset).
- Marketing screenshoty (pro App Store Connect): 5 obrazovek host appky v různých velikostech.
- Update `Keybo/Resources/Assets.xcassets/AppIcon.appiconset/Contents.json` + soubory.

## Závislosti

Žádné code dependencies — jen designerský artefakt.

## Proč ne v v1.0

V1.0 = MVP pro osobní použití. Polish přijde předtím, než půjde na App Store.
