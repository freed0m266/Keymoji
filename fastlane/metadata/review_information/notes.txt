Keymoji requests "Allow Full Access" solely to use the haptic feedback API, which iOS gates behind Full Access for keyboard extensions. The keyboard extension makes no network requests and nothing you type ever leaves the device; the app sends only anonymous, opt-out usage statistics (which settings are used — never content) via TelemetryDeck. Full Access is optional — all typing features (including key click sounds) work without it.

No demo account is needed: the app has no login, accounts, or server. Just add the Keymoji keyboard in Settings → General → Keyboard → Keyboards → Add New Keyboard, then optionally enable Allow Full Access for haptic feedback.

Promotional Plus offers (please note): The app offers an explicit opt-in 30-day Keymoji Plus trial in onboarding and in Settings — the user activates it as a gift and pays zero. It does not bypass in-app purchase: Keymoji Plus is also available via the standard non-consumable IAP, and no payment is taken outside Apple.
