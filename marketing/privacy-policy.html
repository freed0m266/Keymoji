Keybo — Privacy Policy

Last updated: 2 June 2026

1. What we collect

Nothing leaves your device. Keybo is a custom iOS keyboard that runs entirely on your iPhone with no network access, so we — the developers — never receive, transmit, or have any way to see:

  • Text you type or have typed in any app
  • Words, phrases, or patterns from your input
  • Personal information, device identifiers, or usage analytics
  • Network requests of any kind — Keybo has no internet access by design

To make typing faster, Keybo does remember some words you type, stored in a private container on your iPhone so it can suggest completions. This data never leaves your device and we never see it — see Word suggestions below.

2. Why does Keybo ask for "Allow Full Access"?

iOS requires custom keyboards to enable "Allow Full Access" before they can use the haptic feedback and key click sound APIs. This is an iOS sandbox restriction, not a data-collection mechanism. Keybo uses Full Access exclusively for:

  • Haptic feedback (vibration) on key taps
  • Key click sounds when you type

Keybo does not use Full Access for network access, keychain access, or any other purpose. Your keyboard preferences are stored in a shared container between the host app and the keyboard extension using an App Group — that works regardless of Full Access and never involves the network.

3. Third-party services

Keybo uses no third-party services, SDKs, or analytics frameworks.

4. Data sharing

Keybo does not share any data with any party because there is no data to share.

5. Children's privacy

Keybo does not collect data from any user, including children under 13.

6. Changes to this policy

If Keybo's behavior ever changes in a way that affects privacy (e.g. adding an optional cloud sync feature), this policy will be updated and the change will be highlighted in the app's release notes.

Word suggestions

When you type, Keybo can suggest words to complete what you're writing. To make these suggestions better over time, Keybo remembers words you frequently type — words of 3 or more characters from regular text fields, plus email addresses you type in email-typed fields. This data is stored in a private container on your iPhone that only Keybo can read. It is never sent to any server, not even Apple's. You can remove all learned words anytime in Keybo → Settings → Suggestions → Clear learned words.

Keybo never learns from:

  • Password fields, secure text entry fields, or fields marked for one-time codes
  • Credit card number fields
  • Number pads, phone fields, or decimal pads
  • URL bars or web search fields
  • Name fields

Word suggestions can be turned off entirely in Settings → Suggestions.
